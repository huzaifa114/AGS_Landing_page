# Hero background video (optional)

The site includes a **built-in animated motion background** (particles, scan line, grid) — no video file needed.

To use your own looping MP4 instead, place it here as `hero-bg.mp4`.

Recommended if you add a custom file:
- Abstract, soft motion (no logos or competitor branding)
- 10–20 seconds, seamless loop
- Muted palette matching White Whale indigo / cyan aesthetic

If `hero-bg.mp4` is present, it replaces the built-in motion layer automatically.
